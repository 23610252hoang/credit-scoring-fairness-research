## 🔴 企業が「減点」するポイント（避けるべき）

### ❌ 避けるべき説明

```
× 「TensorFlow を使ってディープラーニングで実装しました」
  → 複雑さだけで、実務的価値が低い

× 「精度が 99% 達成できました」
  → 過剰最適化の可能性、汎化性の検証がない

× 「このグラフは何ですか？」（説明できない）
  → 自分の研究を理解していない

× 「公平性指標って何ですか？」
  → 基本的な知識が不足している
```

### ✅ 良い説明

```
✓ 「scikit-learn, XGBoost, SHAP で実装しました」
  → 実務的なツールを選択している

✓ 「Cross-Validation で信頼区間を計算しました」
  → 統計的厳密性がある

✓ 「このグラフは Accuracy vs Fairness です。
   XGBoost が右下に位置して、
   精度と公平性を両立していることを示しています」
  → 自信を持って説明できている

✓ 「Demographic Parity は、グループ間で
   融資OK率が等しいことを意味します」
  → 基本的な知識を理解している
```
