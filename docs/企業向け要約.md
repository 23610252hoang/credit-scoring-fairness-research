# 企業向け要約：与信スコアリングにおける公平性分析

## 📌 研究の核心（1分で理解）

**問題:** 機械学習モデルによる与信スコアリングでは、年齢や性別による**無意識のバイアス**が発生します。

**ソリューション:** 本研究では、以下の3つの貢献を行いました：

1. **バイアスの定量化** - Demographic Parity（DP）と Equal Opportunity（EO）指標で公平性を測定
2. **バイアス原因の特定** - SHAP値を用いて、どの特徴量がバイアスを引き起こしているかを解明
3. **公平性改善手法の検証** - Reweighing法で精度を維持しながらバイアスを52%削減

---

## 🎯 実務的な価値

| 項目 | 結果 |
|------|------|
| **精度維持** | 77.8% (Random Forestでは精度+2%向上) |
| **バイアス削減** | 性別バイアス 52% 削減 |
| **汎化性** | 2つのデータセット間で有効性を検証済み |

## 🔬 技術的なハイライト

```
使用技術:
- 言語: Python 3.11
- MLモデル: Logistic Regression, Random Forest, XGBoost
- 検証方法: Stratified 5-Fold Cross-Validation
- 解釈性: SHAP (SHapley Additive exPlanations)
- 改善手法: Reweighing (前処理ベース)

実験規模:
- データセット: 2つ（German Credit, South German Credit）
- モデル比較: 3種類
- 評価指標: 6つ（精度、AUC、DP_Age、DP_Sex、EO_Age、EO_Sex）
- Cross-Validation: 5-fold × 3モデル = 15実験
```

---

## 📊 主要な発見

### 発見1: 精度と公平性は両立可能
```
❌ 仮説: 精度 ↑ → 公平性 ↓ (トレードオフ)
✅ 結果: XGBoost は最高精度(77.8%) AND 最低バイアス(3.0%) を実現
```

### 発見2: バイアスの根本原因はデータ構造
```
SHAP分析により、以下の特徴量がバイアスの主因と判明:
1. 当座預金残高 (SHAP = 0.79) ← 年齢で大きく異なる
2. 借入金額     (SHAP = 0.51) ← 若年層で構造的に少ない
3. 返済期間     (SHAP = 0.39) ← 年齢と高相関
```

### 発見3: 単発評価は信頼できない
```
Week 1（単発）  : DP_Age = 1.54% ← 過度に楽観的
Week 2（5-fold）: DP_Age = 5.0-6.3% ← 実態を反映
→ 公平性評価には必ず Cross-Validation を使用すべき
```
